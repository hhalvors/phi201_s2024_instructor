Meta-rules for proofs

Substitution

Since our rules of proof are schematic, changing propositional constants (i.e. atomic sentences) in a proof preserves validity.

BUT: an invalid argument can be made valid by substitution! Example: P → Q is invalid, but P → P is valid.

Cut

You don’t need to be able to articulate “cut”, but it is one important way that old proofs can be spliced together to make new proofs. Let’s illustrate it with an example.

Relating proofs and truth tables

Soundness theorem: If there is a proof of the sequent A1, …, An ⊢ B, then the corresponding argument is valid. That is, for any row in their joint truth table, if all of A1, …, An are true, then B is also true.

What it means in practice: If there is a way for all of A1, …, An to be true and B false, then there is no proof of B from A1, …, An. (No correctly written proof with end with B on the main line, and dependencies A1, …, An.)

Completeness theorem: If the argument from A1, …, An to B is valid (in the sense of truth tables), then there is a correctly written proof of the sequent A1, …, An ⊢ B.

What it means in practice: If the truth table for A1, …, An; B says that it’s valid, then there is a proof — no matter how unlikely it might seem!

As a special case: if B is a tautology (true in all cases), then there is a proof of the sequent  ⊢ B. Examples: P ∨ ¬P, P → P, ((P→Q)→P) → P.
